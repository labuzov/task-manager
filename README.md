# Task Manager (pre-alpha)

[Demo](https://labuzov.github.io/task-manager/)

## About this project
This is just a simple task manager. 
Features: authorization, adding, deleting tasks and folders for tasks and a dark theme, of course

## Stack

* ReactJS + Hooks
* React Router
* Redux
* Redux Thunk
* Reselect
* Axios
* Formik
* Yup
* SCSS
